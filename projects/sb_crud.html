Spring Boot REST API To Manage Students Detals

  • Tech Stack: Spring Boot, MySQL, Postman
  • Github URL: Project Link

Developed a REST API project using Spring Boot and MySQL as a backend web service.

Used Spring Web MVC and Spring Data JPA to create RESTful endpoints and perform CRUD operations on MySQL database.

Configured MySQL properties and dependencies in application.properties file and pom.xml file

Applied Spring Boot features such as auto-configuration, dependency injection and starter dependencies to simplify the development process

Tested and documented the REST APIs using Postman.
